# Screenshots Directory

This directory contains screenshots for the README.md documentation.

## Required Screenshots

Please add the following screenshot files to showcase the Calendar component:

- `Month_View.png` - Default month view with events
- `Week_View.png` - Week view layout with time slots
- `Event_Modal.png` - Event creation/editing modal
- `Mobile_View.png` - Mobile responsive design
- `Event_Management.png` - Event management interface
- `Accessibility_Demo.png` - Keyboard navigation demo
- `Storybook_Documentation.png` - Storybook interface
- `Interactive_Demo.png` - Interactive component demo

## Screenshot Guidelines

- Use high-quality PNG format
- Recommended resolution: 1200x800 or higher
- Show the component in action with sample data
- Ensure good contrast and readability
- Include browser UI for context when relevant

## How to Take Screenshots

1. Start the Storybook: `npm run storybook`
2. Navigate to different stories
3. Take screenshots of each view/feature
4. Save them in this directory with the exact names listed above
